# Suggested Commands

## Development
*   **Start Dev Server**: `npm run dev` (Runs `tsx server.ts`)
*   **Build**: `npm run build`
*   **Start Production**: `npm start`

## Quality Assurance
*   **Lint**: `npm run lint` (ESLint)
*   **Test**: `npm test` (Vitest)

## System Utilities (Linux)
*   **File Operations**: `ls`, `cd`, `cat`, `grep`, `find`
*   **Git**: `git status`, `git add`, `git commit`, `git push`

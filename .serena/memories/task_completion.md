# Task Completion Checklist

After completing a coding task, ensure the following:

1.  **Linting**: Run `npm run lint` to check for code style issues.
2.  **Testing**: Run `npm test` to ensure no regressions (if applicable).
3.  **Build**: Run `npm run build` to verify the application builds successfully.
4.  **Verification**: Manually verify the changes in the browser if possible/relevant.
5.  **Cleanup**: Remove any temporary files or debug logs.

# Web Application Stack

This stack installs a complete web application environment consisting of an Nginx web server and a Redis cache.

## Included Services

- [x] nginx-web — Reverse-Proxy + automatische Let's-Encrypt-Zertifikate
- [x] redis-cache — In-Memory-Datenbank für Sessions, Caches, Pub/Sub

## Usage

Select the services you want to install and configure the required variables (Port, Version, etc.) in the next step.
